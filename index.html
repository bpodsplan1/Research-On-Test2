n8n 워크플로우 JSON 파일에는 실제 서버 주소(도메인)가 포함되지 않아 완전 자동 연동은 불가능해요. n8n에서 각 Webhook 노드를 열어 Production URL을 복사해 정확한 주소로 교체해주세요. 또한 내보내신 두 워크플로우 모두 비활성(Inactive) 상태였으니, n8n 화면 우측 상단의 활성화 스위치를 꼭 켜주세요.

모든 요청은 POST 방식으로 JSON 본문을 전송합니다.
